## What is it?
Based on [bevy_cg_editor](https://github.com/BlackPhlox/bevy_cg_editor), `bevy_cursed_editor` is an visual editor for bevy codegen.

## Usage
1. Clone project
2. `cargo run` in the project direcotry
3. Press `Open window` in the top panel and select `Cursed Overview`
4. Here you can see the current/default bevy model, go to cargo run to run the bevy app
5. Alternativly you can go to `File` > `New Project` > And select one of the templates.
6. You can also import/export files in `File` > `Import Json`/`Export Json`
